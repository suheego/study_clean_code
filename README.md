# Introduction

Robert C. Martin의 저서 Clean Code를 읽고 스터디를 한 결과물을 정리했습니다.



# Study Goal

1. 좋은 코드에 대한 기준을 성립할 수 있게 됩니다. 
2. 좋은 코드를 작성하기 위해 어떤 노력이 필요한지 알 수 있게 됩니다.
3. 협업이 용이한 코드를 작성할 수 있게 됩니다.
